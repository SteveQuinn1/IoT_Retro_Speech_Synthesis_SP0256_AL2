MQTT Topics

Software Version
Topic to publish to request for software version of this device
Topic : WFD/SpeechTH/SwVer/Command
Payload : Nothing

Topic to subscribe to, to receive for software version of this device
Topic : WFD/SpeechTH/SwVer/Confirm
Payload : Generic Device, MAC Addr, Filename.ino

WiFi Status
Topic to publish to request for WiFi RSSI Level of this device
Topic : WFD/SpeechTH/1/RSSILev
Payload : Nothing

Topic to subscribe to, to receive for software version of this device
Topic : WFD/SpeechTH/1/RSSILev/Confirm
Payload : dBm of received signal strength

LED Control
Topic to publish to for the red led control signal
Topic : WFD/SpeechTH/1/RedLed/Command
Payload : 0 = Led Off, 1 = Led On

Topic to publish to for the blue led control signal
Topic : WFD/SpeechTH/1/BlueLed/Command
Payload : 0 = Led Off, 1 = Led On

Topic to subscribe to, to receive confirmation of Red/Blue Led control
Topic : WFD/SpeechTH/1/Led/Confirm
Payload : 0 = Led Off, 1 = Red Led On, 2 = Blue Led On, 5 = Error

Hex Phonemes Command
Topic to publish Hex Phonemes to
Topic : WFD/Speech/HexPhonemes/Command
Examples
Steven Quinn,   Payload : 370D130123020B0308002E0C000B00
System Ready, Payload : 37000C37000D000710000E07210C1400
Hello World,      Payload : 1B07072D0035032E01342D001500

Topic to subscribe to, to receive confirmation the hex phonemes have been spoken.
Topic : WFD/Speech/HexPhonemes/Confirm
Payload :  0 = Done, 1 = Busy, 5 = Error, 6 = Error, uneven number of hex phonemes

Straight Phonemes Command
Topic to publish Phonemes codes to.
Topic : WFD/SpeechTH/1/Phonemes/Command
Examples
Steven Quinn,   Payload : SS,TT2,IY,PA2,VV,PA3,NN1,PA4,KK3,PA1,WW,IH,PA1,NN1,PA1
System Ready, Payload : SS,PA1,IH,SS,PA1,TT2,PA1,EH,MM,PA1,RR1,EH,DD2,IH,EY,PA1
Hello World,      Payload : HH1,EH,EH,LL,PA1,OW,PA4,WW,PA2,ER2,LL,PA1,DD1,PA1

Topic to subscribe to, to receive confirmation the phonemes have been spoken.
Topic : WFD/SpeechTH/1/Phonemes/Confirm
Payload : 0 = Done, 2 = Busy, 5 = Error, bad/unrecognised phoneme

Fixed Phrase Phonemes Command
Topic to publish Phonemes codes to, to speak a fixed phrase. Payload is index number of phrase to speak. There are currently 3 fixed phrases.
Topic : WFD/SpeechTH/1/FixedPhrase/Command
Payload : 0 = Steven Quinn, 1 = System Ready, 2 = Hello World

Topic to subscribe to, to receive confirmation the fixed phrases have been spoken.
Topic : WFD/SpeechTH/1/FixedPhrase/Confirm
Payload : 0 = Done, 3 = Busy, 5 = Error, unrecognised phrase index

Direct Word Phonemes Command
This is ToDo, as it requires a MySQL database connection
Topic to publish to for the speaking of a single word. Word is looked up in database
Topic : WFD/SpeechTH/1/Word/Command
Payload : Words...

Topic to subscribe to, to receive confirmation the words have been spoken.
Topic : WFD/SpeechTH/1/Word/Confirm
Payload : 0 = Done, 4 = Busy, 5 = Error, unrecognised word, not is database
